Marketing playbook for growing the college basketball Twitter presence tied to Spread Eagle. Start with follower growth via consistently insightful posts, then steer engaged users toward the app once trust is built.

Contents
- twitter_playbook.md — audience, voice, tweet formats, posting cadence, daily workflow
- pillar_library.md — content pillars with examples
- hashtag_research.md — how to stay on top of trending CBB topics
- content_calendar.md — simple weekly calendar template
- kpi_scorecard.md — core metrics to track and improve

Quick start
1) Pick 1-2 pillars for today and draft 4-6 tweets (mix of insights, tonight’s picks, and educational context).  
2) Run hashtags using the hashtag_research checklist.  
3) Schedule/queue with 1-hour spacing; leave 2 slots open for live/reactive moments.  
4) After posting, engage: reply to analysts, quote-tweet news, answer questions.  
5) End-of-day: log performance in kpi_scorecard and note which angles resonated.
